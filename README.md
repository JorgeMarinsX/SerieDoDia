# 📺 Série do Dia

> O fim definitivo de todas as brigas sobre o que assistir hoje.

**Série do Dia** é uma aplicação web estática que sorteia uma série aleatória — ideal para aqueles momentos em que ninguém consegue decidir o que ver.

Clone de [Filme do Dia](https://github.com/JorgeMarinsX/FilmeDoDia), mas para séries de TV.

---

## ✨ Funcionalidades

- Sorteio aleatório de séries via [TMDB API](https://www.themoviedb.org/)
- Exibe pôster, título, ano de estreia, nota, número de temporadas e sinopse
- Mostra onde assistir (provedores de streaming no Brasil via JustWatch)
- Layout responsivo e mobile-first com Bootstrap 5
- Deploy automático via GitHub Pages

---

## 🛠️ Tecnologias

| Tecnologia | Uso |
|---|---|
| HTML5 + JavaScript (vanilla) | Estrutura e lógica da aplicação |
| [Bootstrap 5.3](https://getbootstrap.com/) | Estilização e componentes responsivos |
| [TMDB API](https://developer.themoviedb.org/) | Fonte dos dados de séries |
| GitHub Actions | CI/CD e injeção de variáveis de ambiente |
| GitHub Pages | Hospedagem estática |
